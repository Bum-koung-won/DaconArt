# DaconArt
데이콘 경진대회 / 예술 작품을 화가 별로 분류하는 AI 모델 개발

▪ 제공 받은 화가 및 예술 작품 이미지 데이터 확인 및 분석

▪ 화가(라벨) 별 예술작품 분류, Augmentation 이용 데이터 증강

   -> ImageDataGenerator
   
▪ Image Classification 모델별 학습 및 성능 향상을 위한 상세 조정

   -> VGG16, ResNet50, MobileV2, EfficientNetV2L
   
▪ 최종 모델 선정 후 예측

   -> EfficientNetV2L
